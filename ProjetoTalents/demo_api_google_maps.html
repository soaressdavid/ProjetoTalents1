Como Configurar a API Real
  1. Acesse o Google Cloud Console
  2. Crie um projeto e habilite as APIs necessárias
  3. Gere uma chave de API
  4. Configure no arquivo config/google_maps.php
  5. Siga as instruções em CONFIGURAR_API_GOOGLE_MAPS.md
Importante

A API do Google Maps tem custos após o limite gratuito. Consulte a documentação oficial para mais informações sobre preços e limites.
